# Recipe Management REST API

REST API сервис для управления рецептами.

Основа проекта взята из лабы 3. Теперь сервис, который работает с рецептами и ингредиентами, сохраняет информацию в MongoDB

## Стек

- C++
- POCO
- JWT
- Docker
- PostgreSQL
- MongoDB

## Запуск

docker compose up --build

## Swagger:

http://localhost:8080/swagger.yaml

## API docs:

http://localhost:8080/docs
