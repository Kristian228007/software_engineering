# Recipe Management REST API

REST API сервис для управления рецептами.

Основа проекта взята из лабы 2. Теперь вместо хранения данных in memory используется бд PostgreSQL

## Стек

- C++
- POCO
- JWT
- Docker
- PostgreSQL

## Запуск

docker compose up --build

## Swagger:

http://localhost:8080/swagger.yaml

## API docs:

http://localhost:8080/docs
